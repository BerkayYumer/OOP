# OOP:# Футболна система (C#)

## Кратко описание

Този проект представлява проста конзолна футболна система, написана на езика C#, която демонстрира основни принципи на обектно-ориентираното програмиране.

## Какво прави проектът

Проектът позволява:

* създаване на футболни играчи с име, номер и позиция;
* създаване на футболни отбори и добавяне на играчи към тях;
* извеждане на списък с играчите в даден отбор;
* създаване на мач между два отбора;
* задаване и показване на резултат от мача в четим текстов формат.

## Как се стартира проектът (Visual Studio)

1. Отворете Visual Studio.
2. Изберете **Open a project or solution**.
3. Отворете папката с проекта или `.sln` файла.
4. Уверете се, че проектът е зададен като **Startup Project**.
5. Стартирайте програмата с клавиш **F5** или бутона **Start**.

## Използвани технологии

* Програмен език: **C#**
* Среда за разработка: **Visual Studio**
* Тип приложение: **Конзолно приложение**
